呃，其實很接近了

比較直接的想法就是

`exp(x, n) = exp(x, n - 1) * x;`


就這樣，你已經寫完一半了，n == 0 的那個遞迴邊界

只是你要改 else 的部分

BTW，之後你可能會學到這方法其實很慢

因為要算 n 次乘法，但可以不用那麼多

是指遞迴很慢嗎?

不算遞迴的問題，是計算方式的問題

x^n 雖然可以從 x^(n - 1) 推出

但也可以從 x^(n / 2) 推出來
